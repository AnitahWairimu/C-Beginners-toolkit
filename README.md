## Project Description
This project presents a beginner-friendly toolkit designed to introduce new programmers to C# (C-Sharp). The toolkit uses simple, runnable console applications to demonstrate fundamental programming concepts such as variables, control structures, methods, and basic object-oriented programming. The goal is to reduce the learning curve for beginners by providing clear examples and structured guidance.

## Technologies Used
- C#
- .NET SDK
- Visual Studio Code
- Console-based Application

## How to Run the Project
1. Install the .NET SDK.
2. Install Visual Studio Code.
3. Clone or download the repository.
4. Open the project folder in Visual Studio Code.
5. Run the application using the terminal.
